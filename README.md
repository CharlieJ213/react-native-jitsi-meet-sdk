# react-native-jitsi-meet-sdk

React native wrapper for Jitsi Meet SDK Library

## Installation

```sh
npm install react-native-jitsi-meet-sdk
```

## Usage

```js
import { multiply } from "react-native-jitsi-meet-sdk";

// ...

const result = await multiply(3, 7);
```
